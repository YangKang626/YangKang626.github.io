    # 二、win7游戏全屏

    # 1、在开始运行输入regedit打开注册表

    # 2、定位到

    HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\GraphicsDrivers\Configuration
    1

    # 3、然后右键点击Configuration，选择查找，输入Scaling，在右框内即可看到 scaling，右键scaling选择修改，将数值改为3即可，原值为4。
